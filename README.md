## Installation

1. Install [Java](https://www.java.com/en/download).

2. Clone this project to your local hard drive: `git clone https://github.com/Adamouization/SRPN`

3. Run the following commands inside the project directory:

```
javac SRPN.java
java SRPN.java
```
